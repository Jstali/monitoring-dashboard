# Monitoring Dashboard

Modern enterprise monitoring dashboard built with React.js and Tailwind CSS.

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm start
```

The application will open at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Features

- Sticky header with branding
- Fixed sidebar navigation with active states
- Dashboard title section with icon
- Quick access cards grid (10 cards)
- Live queue manager status section (6 cards)
- Responsive layout
- Smooth hover animations
- Modern enterprise design
